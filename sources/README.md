# Sources

This public demo intentionally does not include private local source packets.

During the Thaw investigation, one private UI/UX research packet was used only as a terminology and framing reference. The raw packet, local source paths, and private source map are not published here.

Public-safe derived material from that packet is kept in `analysis/ui-ux-negligence-frame.md`.

This mirrors the HARE Trail source-boundary rule: preserve source context when possible, but do not publish private work, personal history, company context, credentials, or local paths just to make a demo self-contained.
