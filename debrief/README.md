# Debrief Demo

Эта папка — public-demo копия Thaw-specific debrief material.

В реальной раскладке HARE Trail эти слои разнесены:

- canonical session debriefs живут в `session-debriefs/`;
- extracted reusable lessons живут в `LESSONS.md`;
- task evidence и drafts живут в `work-artifacts/<task>/`.

В этом public example repo debrief и extracted lessons намеренно colocated здесь, чтобы читатель мог увидеть полный loop в одном месте без доступа к private HARE Trail data repository.

## Files

- `session-debrief.md` - sanitized debrief по Thaw problem-notes session.
- `lessons.md` - extracted Thaw-specific lessons и локальная таблица статистики ошибок.
